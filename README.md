# Healthcare Readmissions & Cost Analysis Project

## Objective
Analyze patient readmissions and cost patterns using SQL to evaluate the impact of care coordination and patient risk levels.

## Dataset
Simulated healthcare dataset including:
- Patient demographics
- Hospital admissions
- Post-discharge follow-up data

## Tools Used
- PostgreSQL
- Microsoft Excel

## Analysis Process

1. Data was extracted and structured using PostgreSQL
2. SQL queries were used to analyze:
   - Readmission rates
   - Impact of follow-up care
   - Cost differences by patient risk level
3. Query results were exported as CSV files
4. Excel was used to visualize results through bar charts:
   - Follow-up care vs readmission rates
   - High-risk vs low-risk patient cost comparison
5. Insights were summarized to support healthcare decision-making

## Key Analyses
1. Readmission rate analysis
2. Follow-up care impact on readmissions
3. Cost differences by patient risk level

## Key Findings
- Patients with 7 day follow up had lower readmission rates
- High-risk patients had higher average hospitalization costs
- Care coordination is associated with improved patient outcomes

## Recommendations
- Improve 7 day follow-up compliance for high-risk patients
- Strengthen discharge planning processes
- Focus resources on high-cost patient groups

## Skills Demonstrated

- SQL data analysis using PostgreSQL (CREATE, JOINs, GROUP BY, aggregations)
- Cohort analysis to evaluate patient outcomes (follow-up care vs readmission rates)
- Healthcare cost analysis and risk stratification
- Data cleaning and structuring for relational datasets
- Exporting query results for Excel based visualization
- Translating raw data into actionable healthcare insights
- Basic healthcare performance evaluation (readmissions, cost drivers, care coordination impact)
